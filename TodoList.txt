[ ] 實作 AdminRepository（ADO.NET）
    - 位置：ISpanShop.Repositories/AdminRepository.cs

    - SQL GetAllAdmins：
        SELECT U.Id          AS UserId,
               U.Account,
               U.Email,
               U.RoleId,
               R.RoleName,
               U.AdminLevelId,
               AL.LevelName  AS AdminLevelName,
               U.IsBlacklisted,
               U.IsFirstLogin,
               U.CreatedAt
        FROM   Users U
        JOIN   Roles R       ON U.RoleId = R.Id
        LEFT JOIN AdminLevels AL ON U.AdminLevelId = AL.Id
        WHERE  R.RoleName IN ('SuperAdmin', 'Admin')
        ORDER BY U.CreatedAt DESC

    - SQL GetSelectableAdminLevels
      （下拉選單來源，排除超級管理員）：
        SELECT Id AS AdminLevelId, LevelName, Description
        FROM   AdminLevels
        WHERE  Id != 1
        ORDER BY Id

    - SQL GetAdminByAccount：
        SELECT U.Id          AS UserId,
               U.Account,
               U.Email,
               U.Password    AS PasswordHash,
               U.RoleId,
               R.RoleName,
               U.AdminLevelId,
               AL.LevelName  AS AdminLevelName,
               U.IsBlacklisted,
               U.IsFirstLogin
        FROM   Users U
        JOIN   Roles R       ON U.RoleId = R.Id
        LEFT JOIN AdminLevels AL ON U.AdminLevelId = AL.Id
        WHERE  U.Account = @Account
          AND  U.IsBlacklisted = 0

    - SQL GetNextAdminSequence：
        SELECT ISNULL(MAX(Id), 0) + 1 FROM Users

    - SQL CreateAdmin：
        INSERT INTO Users
            (RoleId, Account, Password, Email,
             AdminLevelId, IsConfirmed,
             IsBlacklisted, IsFirstLogin, CreatedAt)
        VALUES
            (@RoleId, @Account, @PasswordHash, @Email,
             @AdminLevelId, 1,
             0, 1, GETDATE())

    - SQL DeactivateAdmin（軟刪除）：
        UPDATE Users
        SET IsBlacklisted = 1, UpdatedAt = GETDATE()
        WHERE Id = @UserId

    - SQL IsAccountExists：
        SELECT COUNT(1) FROM Users WHERE Account = @Account

    - SQL ChangePassword：
        UPDATE Users
        SET Password = @PasswordHash, UpdatedAt = GETDATE()
        WHERE Id = @UserId

    - SQL SetFirstLoginComplete：
        UPDATE Users
        SET IsFirstLogin = 0, UpdatedAt = GETDATE()
        WHERE Id = @UserId

[ ] 實作 AdminService
    - 位置：ISpanShop.Services/AdminService.cs

    - CreateAdmin 邏輯：
        1. 確認 AdminLevelId 不為 1（不可直接新增超級管理員）
        2. 取得流水號
           int seq = _adminRepository.GetNextAdminSequence()
        3. 組成帳號與 Email
           string account = $"ADM{seq:D3}"
           string email   = $"{account}@ispan.com"
        4. 確認帳號不重複（IsAccountExists）
        5. 角色固定為 Admin
           int roleId = 1   
        6. Hash 初始密碼
           string passwordHash = BCrypt.HashPassword(dto.Password)
        7. 呼叫 Repository.CreateAdmin(
               account, email, passwordHash, roleId, dto.AdminLevelId)
        8. 回傳 GeneratedAccount 供畫面顯示

    - DeactivateAdmin 邏輯：
        1. userId != currentUserId（不可停用自己）
        2. 若目標是超級管理員，檢查超級管理員數量 > 1
           SELECT COUNT(1) FROM Users U
           JOIN AdminLevels AL ON U.AdminLevelId = AL.Id
           WHERE AL.Id = 1 AND U.IsBlacklisted = 0
        3. 呼叫 Repository.DeactivateAdmin(userId)

    - VerifyLogin 邏輯：
        1. 呼叫 GetAdminByAccount(account)
        2. 帳號不存在 → 回傳 null
        3. BCrypt.Verify(password, admin.PasswordHash)
        4. 驗證成功 → 回傳 AdminDto（含 IsFirstLogin）
        5. 驗證失敗 → 回傳 null

    - ChangePassword 邏輯：
        1. NewPassword == ConfirmPassword
        2. 新密碼至少 8 碼且含英文與數字
        3. Hash 新密碼
        4. 呼叫 Repository.ChangePassword()
        5. 呼叫 Repository.SetFirstLoginComplete()
