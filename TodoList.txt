[V]【後台會員管理實作】

    1: 定義資料傳輸物件 (ISpanShop.Models)
        目的：對應 Admins 資料表與權限顯示
        [ V] 建立 AdminsDto 與 AdminsRoleDto
        - 位置：ISpanShop.Models/DTOs/AdminsDtos.cs
        - 內容：AdminsDto (AdminId, Account, Email, RoleId, RoleName)
        - 內容：AdminsRoleDto (RoleId, RoleName) - 用於下拉選單

  
    2：資料存取層 (ISpanShop.Repositories)
        目的：直接操作 Admins 資料表 (ADO.NET)
        [V] 建立介面 IAdminRepository
            - 位置：ISpanShop.Repositories/Interfaces/IAdminRepository.cs
            - 方法：
                IEnumerable<AdminDto> GetAllAdmins()
                bool UpdateAdminRole(int adminId, int roleId)
        [V] 實作 AdminRepository
            - 位置：ISpanShop.Repositories/AdminRepository.cs
            - SQL GetAllAdmins：
                SELECT A.Id AS AdminId, A.Account, A.Email,
                       A.RoleId, R.RoleName
                FROM Admins A
                JOIN Roles R ON A.RoleId = R.Id
            - SQL UpdateAdminRole：
                UPDATE Admins SET RoleId = @RoleId WHERE Id = @AdminId
        [V] 建立介面 IAdminRoleRepository（若角色列表需獨立查詢）
            - 位置：ISpanShop.Repositories/Interfaces/IAdminRoleRepository.cs
            - 方法：
                IEnumerable<AdminRoleDto> GetAllRoles()
        [V] 實作 AdminRoleRepository
            - 位置：ISpanShop.Repositories/AdminRoleRepository.cs
            - SQL GetAllRoles：
                SELECT Id AS RoleId, RoleName FROM Roles
