## (b) 打折活動卡片顯示內容

**限時特賣 / 限量搶購 (flashSale / limitedBuy)**

顯示元素：
1. 活動折扣價（大紅色 #dc2626）
2. 原價刪除線（灰色）
3. 折扣標籤（如「9折」，淺紅背景）
4. 已售數量（右側灰色小字）
5. 左上角活動類型標籤（flashSale=紅色 / limitedBuy=紫色）

視覺範例：
`
┌────────────────────────┐
│ [限時特賣]        商品圖│
│                        │
└────────────────────────┘
 商品名稱（最多兩行）
 ,310              已售 1234
 ,900  [9折]
 ^^^^^^   ^^^^
 刪除線   折扣標籤
`

## (c) 滿額折扣卡片顯示內容

**滿額折扣 (discount)**

顯示元素：
1. 原價（蝦皮橘 #EE4D2D）
2. 已售數量（右側灰色小字）
3. 滿額提示（橘色小字，如「滿1000折100」）
4. 左上角活動類型標籤（橘色 #f97316）

視覺範例：
`
┌────────────────────────┐
│ [滿額折扣]        商品圖│
│                        │
└────────────────────────┘
 商品名稱（最多兩行）
 ,900            已售 1234
 滿1,000折100
`

## 技術細節

### 安全性
- 所有 promotion 相關的顯示都加上 v-if 判斷
- 避免 null reference 錯誤
- 使用可選鏈運算符 (!) 處理必定存在的欄位

### 價格格式化
- 使用既有的 formatPrice() 工具函數
- 確保千位分隔符正確顯示

### 折扣計算
- 折扣標籤：Math.floor(discountPercent / 10)
- 例如 90% → 9折、85% → 8折

### 響應式設計
- 卡片保持正方形圖片 (1:1)
- 價格區域有最小高度，避免無活動時高度不一致
- hover 效果維持不變
