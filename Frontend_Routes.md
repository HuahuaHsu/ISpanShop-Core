# ISpanShop 前台功能與路由對照表

本文件記錄了 `ISpanShop-Frontend` 目前已開發的功能頁面及其對應的網址路徑。

## 🌐 基礎頁面
| 功能名稱 | 網址路徑 (URL) | 說明 |
| :--- | :--- | :--- |
| **首頁** | `/` | 網站入口。 |
| **關於我們** | `/about` | 關於網站的相關資訊。 |

## 🛒 購物與結帳
| 功能名稱 | 網址路徑 (URL) | 說明 |
| :--- | :--- | :--- |
| **購物車** | `/cart` | 檢視購物車商品、調整數量與進入結帳流程。 |

## 👤 會員中心 (個人專區)
| 功能名稱 | 網址路徑 (URL) | 說明 |
| :--- | :--- | :--- |
| **我的訂單** | `/member/orders` | 檢視個人所有的歷史訂單記錄與狀態。 |
| **訂單詳情** | `/member/order/:id` | 檢視特定訂單的詳細內容 (`:id` 為訂單編號)。 |

## 🏪 賣家中心 (Store)
| 功能名稱 | 網址路徑 (URL) | 說明 |
| :--- | :--- | :--- |
| **賣場數據概覽** | `/member/store-dashboard` | 賣家檢視賣場業績、訂單趨勢等統計數據。 |

---
**提示：**
- 前台專案運行時，通常預設網址為 `http://localhost:5173/` (視 Vite 設定而定)。
- 開發時請確保後端 API (`ISpanShop.MVC`) 已啟動，以利資料正常抓取。
