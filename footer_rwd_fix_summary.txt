## (a) 保障區塊的實際 class 名和修改後排版

### 實際 class 名稱

**容器：**
- \.footer-top\ - 保障區塊外層容器（深色背景）
- \.footer-features\ - 保障功能 grid 容器

**項目：**
- \.feature-item\ - 單個保障項目
- \.feature-item :deep(.el-icon)\ - Element Plus 圖標
- \.feature-item h4\ - 保障標題（如「全台快速配送」）
- \.feature-item p\ - 保障描述（如「滿千免運費」）

速 款 賞 服
... (擠在一起)
`

## (b) Footer 的實際 class 名和修改後排版

### 實際 class 名稱

**容器：**
- \.footer\ - Footer 最外層容器
- \.footer-inner\ - Footer 主內容 grid 容器

**各欄位：**
- \.footer-col\ - 單欄容器
- \.footer-brand\ - 品牌欄（第一欄）
- \.footer-logo\ - Logo 區塊
- \.footer-logo-img\ - Logo 圖片

**內容元素：**
- \.footer-col h4\ - 欄位標題
- \.footer-col a\ - 連結
- \.footer-col p\ - 段落文字
- \.footer-bottom\ - 底部版權區
